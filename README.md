# PlantIQ: Crop Disease Detection System

**Overview**  
For my university Final Year Project, I designed and developed PlantIQ, a mobile-first crop disease detection system. The end-to-end architecture pairs a Flutter mobile application with a FastAPI backend to deliver real-time diagnostic results. At its core, I trained and deployed a custom deep learning classification model to accurately identify plant diseases from image data, creating a practical, AI-driven agricultural solution.

**Tech Stack**
* **Frontend:** Flutter
* **Backend:** FastAPI
* **Machine Learning:** Custom Deep Learning Classification Model
